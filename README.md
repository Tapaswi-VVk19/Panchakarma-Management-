# 🌿 Panchakarma Management System

A modern full-stack web application designed to streamline the management of Ayurvedic Panchakarma clinics. The system provides dedicated portals for Patients, Practitioners, and Administrators, enabling efficient appointment scheduling, treatment tracking, diet plan management, notifications, and patient care.

---

## 🚀 Features

### 🔐 Authentication
- Secure login system
- Role-based access control
- Patient, Practitioner, and Admin accounts

### 👤 Patient Portal
- Dashboard with treatment overview
- Request appointments
- View upcoming sessions
- Track treatment progress
- Access personalized diet plans
- Receive notifications and alerts
- Submit feedback

### 👨‍⚕️ Practitioner Portal
- Daily schedule management
- Patient management dashboard
- View upcoming appointments
- Reassign treatment sessions
- Update diet plans
- Send notifications and emergency alerts
- Track patient progress

### 🏥 Admin Portal
- Manage patients and practitioners
- Monitor appointments
- Manage treatment packages
- View clinic statistics
- Configure system settings
